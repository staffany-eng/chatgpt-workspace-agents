# Help Article Generator Skill

This folder contains a reusable Codex skill for creating/updating StaffAny help articles.

## Files

- `SKILL.md`: main skill instructions
- `references/help-article-skeleton.md`: article structure starter

## How To Share With Team

1. Share this folder:
   - `/Users/vanessakwa/Documents/Codex/VK New Workspace /help-article-generator`
2. Teammate copies it into their skills directory:
   - `$CODEX_HOME/skills/help-article-generator`
3. Teammate can trigger it by name:
   - `help-article-generator`

## Expected Behavior

- Supports `Create` and `Update` flows.
- Uses `Mobile/Web` platform labels.
- Applies the agreed article structure, list formatting, and FAQ style.
- Defaults update scans to English help center (`/en/`) first.
