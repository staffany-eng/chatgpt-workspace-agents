# Plan Agent From Research Playbook

Use this when turning the corpus into a ChatGPT workspace-agent plan.

## Steps

1. Read `docs/product-compass.md`.
2. Read `research/wiki/index.md`, `weights.md`, and `decisions.md`.
3. Identify which agent surface is affected: instructions, apps/tools, skills, files, memory, schedules, or channels.
4. Separate official ChatGPT behavior from OpenClaw/Hermes analogies.
5. Use weight `4-5` evidence as planning guidance.
6. Keep weight `1-3` evidence as prompts, caveats, or follow-up questions.
7. If the answer creates reusable cross-source learning, update or create a synthesis page.
8. Promote stable guidance to `research/wiki/decisions.md` only after it is supported by strong evidence.
