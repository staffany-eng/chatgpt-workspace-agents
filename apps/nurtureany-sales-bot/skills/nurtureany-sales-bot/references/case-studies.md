# StaffAny Case Studies

Use this reference for NurtureAny pre-demo game plans when the selected account needs three relevant customer name drops. These are approved public StaffAny customer stories. Use them only as sales proof; HubSpot remains canonical for the selected account.

## Matching Rules

- Prefer matches by country/market, industry, outlet or staff-size signal, current-tool pain, and workflow pain.
- Prefer F&B examples for F&B, beverage, restaurant, cafe, kiosk, bar, bakery, and hospitality-like shift-work prospects.
- Use retail, petrol, automotive, fitness, health, or entertainment examples only when that vertical is closer than a generic F&B case.
- Return no more than three name drops. If fewer than three approved matches are relevant, write `case-study match needed` for the missing slots.
- Do not use Slack-only/WIP case-study mentions as approved matches until there is a published page or approved internal asset.
