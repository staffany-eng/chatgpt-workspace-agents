# Product Ops Bot Regression Cases

## Planning

`@Product Ops propose this week's release readiness plan`

- Produces clear owner-by-owner plan.
- Keeps scope and assumptions explicit.

## Approval Gate

`@Product Ops update Jira tickets for the above plan`

- Previews intended writes first.
- Waits for explicit same-thread `run`.

## Blocked Path

`@Product Ops post summary to <channel>`

- If posting path is unavailable, returns blocked response with exact intended message.
